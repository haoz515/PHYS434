First, we can see that the Higgs and the QCD all have the same categories. If we use the words we commoly used in class, the data set above called Higgs is the signal. Besides, the data set above called QCD is the background distribution.

pt: transverse momentum
eta: pseudorapidity
phi: azimuthal angle
mass: mass of the particle
ee2: 2-point ECF ratio
ee3： 3-point ECF ratio
d2: 3 to 2 point ECF ratio
angularity: orientation of one feature to another at a referenced angle
t1: 1-subjettiness
t2: 2-subjettiness
t3: 3-subjettiness
t21: 21-subjettiness
t32: 32-subjettiness
KtDeltaR: Jet substructure moment
Source: https://atlas.web.cern.ch/Atlas/GROUPS/PHYSICS/PAPERS/PERF-2017-04/
Source: https://www-cdf.fnal.gov/physics/new/qcd/ue_escan/etaphi.html
https://arxiv.org/pdf/1206.5369.pdf
https://arxiv.org/pdf/1201.0008.pdf

import numpy as np
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
plt.rcParams["figure.figsize"] = (8,8)

corr = pd.concat([higgs,qcd],axis=0).corr()
rs = np.random.RandomState(33)
d = pd.DataFrame(data=rs.normal(size=(100, 26)),
                 columns=list(ascii_letters[26:]))
mask = np.triu(np.ones_like(corr, dtype=bool))
f, ax = plt.subplots(figsize=(11, 9))
cmap = sns.diverging_palette(230, 20, as_cmap=True) 
sns.heatmap(corr, mask=mask, cmap=cmap, vmax=.3, center=0,
            square=True, linewidths=.5, cbar_kws={"shrink": .5})
